URL Parameters Guide

Available URL Parameters

?site=SITE_KEY

Force load a specific site, bypassing IP detection.

?site=car ?site=ems

?audience=staff|student

Set the view to staff or student. Default is student.

?audience=staff ?audience=student

?view=minimal|icons-only

Control how much information is displayed. You can also use the radio buttons at the bottom of the page to change the view - the URL will update automatically!

?view=minimal - Hides descriptions
?view=icons-only - Shows only icons

?apps=app1,app2,app3

Show only specific applications.

?apps=email,teams,calendar

Combining Parameters

Use & to combine multiple parameters:

?site=ems&audience=staff&view=minimal

Examples

Student kiosk:
?audience=student&view=icons-only

Staff at specific site:
?site=car&audience=staff

Minimal view with specific apps:
?view=minimal&apps=email,teams,library
